🎓 Feedback Momenten Setup

Verbind met Google Sheets in enkele eenvoudige stappen

Stap 1: Google Cloud Setup (eenmalig)

Om te verbinden met Google Sheets heb je een Google Cloud project nodig. Dit is gratis en duurt slechts enkele minuten.

Snelle Setup:
  1. Ga naar Google Cloud Console
  2. Maak een nieuw project (of selecteer bestaand)
  3. Klik "+ CREATE CREDENTIALS" → "OAuth client ID"
  4. Indien gevraagd, configureer OAuth consent screen (External, voeg je email toe als test user)
  5. Application type: "Web application"
  6. Voeg Authorized JavaScript origin toe: http://localhost
  7. Klik "Create" en kopieer je Client ID
Schakel ook de Sheets API in:
  1. Ga naar Sheets API
  2. Klik "Enable"

Stap 2: Aanmelden met Google

Meld je aan zodat de app een Google Sheet voor je kan aanmaken.

Stap 3: Google Sheet Aanmaken

Klik op de knop hieronder om een nieuwe Google Sheet aan te maken met de juiste structuur.

Stap 4: Backend Script Toevoegen

Open je nieuwe sheet en voeg het Apps Script toe:

📄 Open Je Google Sheet
In je Google Sheet:
  1. Ga naar Extensions → Apps Script
  2. Verwijder bestaande code
  3. Plak de code hieronder en sla op
  4. Klik Run → initializeSheets (geef toestemming)
  5. Klik Deploy → New deployment
  6. Type: Web app, Execute as: Me, Access: Anyone
  7. Kopieer de deployment URL en plak hieronder
// Klik "Code Kopiëren" hierboven om de volledige Apps Script te kopiëren
